我把 Qwen 3.7 MAX 网页模型，
接进了小龙虾和 Hermes。

网页模型也能调用工具干活了？

但先说边界：
别拿它硬跑复杂编程任务。

它更适合轻中度任务：
整理资料、查信息、简单工具调用、批量小活。

用 WebLLM Gateway，
登录网页账号，自动检测模型，
复制地址、Key 和模型。

填到小龙虾、Hermes 这类兼容客户端里，
就能开始用。

重点不是多一个聊天窗口。
而是网页模型也能走工具调用链路。

Gateway 负责翻译：
客户端把工具说明发过来，
它翻译给网页模型。
模型要工具时，
再翻译回标准工具调用。

真正执行工具和权限确认的，
还是客户端。

所以它适合养虾养马、低成本试错，
省一部分 API Token。

复杂工程长链路就别硬上。

项目已开源：
WebLLM Gateway。
